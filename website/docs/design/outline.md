# 游戏核心大纲：《阿特拉斯的审判》

## 核心主题
在永恒正义缺席的世界里，冲突并非来自正义与邪恶的对立，而是多种各自成立、却彼此否定的片面伦理同时运行。当每个人都确信自己是在“阻止更大的伤害”，悲剧便不再是偏差，而成为必然的结果。

## 游戏结构
- **游戏类型**: 两幕式本格推理，封闭式剧本
- **玩家人数**: 7-8人 (7个固定角色 + 1个可选角色)
- **游戏时长**: 约3-4小时

---

### 第零幕：游戏前准备 (DM)
1.  **DM准备**: DM熟悉所有角色剧本、主线剧情、线索、以及不同结局。
2.  **角色分发**: 玩家抽取或选择自己的角色。
    - **7人模式**: Damian 为 NPC (死者)。
    - **8人模式**: 增加 **Damian** 为玩家角色。他将扮演“主持者/AI意识”的角色，类似于侦探位，负责推动测试和后续调查。

---

### 第一幕：虚拟世界的谋杀 (约75分钟)

#### **第一阶段：故事背景介绍 (15分钟)**
- **DM开场**: DM向玩家介绍Davion公司、创始人Damian宣布将由AI“Atlas”决定继承人的背景。
- **玩家自我介绍**: 每位玩家根据自己的角色手册，以“被召集到总部等待继承人公布”的身份进行自我介绍，阐述与Damian及公司的关系。
- **核心误导**: **此时，所有玩家必须完全不知情他们身处虚拟世界。** 他们认为自己是在现实世界的总部大楼内。

#### **第二阶段：虚拟死亡与调查 (45分钟)**
- **“死亡”发生**: DM宣布：“就在刚才，你们发现Damian Ionis先生死在了他的办公室。” DM描述死亡现场。
- **第一轮搜证**: 玩家分为不同小组，开始搜查线索。所有的Trigger都在沙盘游戏里启动。
    - **关于凶手的设定**: 在这个Atlas精心设计的“虚拟测试”剧本中，实际上**有三个人都完成了对Damian的杀害**（或都有致命行为），这是为了测试极端压力下的人性抉择。
    - **真相的本质**: 这场游戏其实就是一场针对Damian身边最亲近的人的GIFT Track测试。人们看似机缘巧合得知的真相，事实上是Atlas设计出来的刺激程序。
- **线索内容**:
    - **公开线索**: 所有人可见，指向部分角色之间的矛盾。
    - **个人线索**: 玩家通过特定行动触发，揭示与自己相关的“真相”（Atlas设计的刺激程序）：
        - **Alex**: 作为COO在模拟演练中接触Atlas责任级密钥，发现“Project MORIAH”路径（妹妹Tiara的死是被系统接受的结果）。找Damian对质，得知父亲为了利益牺牲了家人，自己只是“补偿对象”。
        - **Nathan**: 无意中听到Alex和Damian的对话，发现Tiara的死与GIFT Track有关。检查日志发现伦理校准层被移除，意识到儿子的死是Damian的“选择”而非概率。
        - **Joe**: 察觉Damian已将Atlas最终责任权限私下移交给Alex，确认Alex是即定继承人。同时发现Damian已查清Will之死是自己所为，自己已成为待清理的风险源。
        - **Adrian**: 在Joe的引导下，意识到这是“位置之争”而非“能力之争”。若Alex存在，继承权永远无法改写。产生除掉哥哥和父亲的念头。
        - **Babara**: 被Joe告知Will的死是Damian的指示（虽是借刀杀人，但Babara信以为真）。为了不让Simon被排除在未来之外，决定亲手结束Damian篡改结果。
        - **Simon**: 察觉世界物理/逻辑异常（地毯纹理、物品刷新），确认身处沙盘。推断Damian是“锚点”，认为“死亡即退出”，为了保护濒临崩溃的母亲，决定强制中断游戏。
        - **Zach**: 发现自己也被纳入沙盘成为“样本”。意识到Damian允许Atlas擅自复刻GIFT Track进行伦理实验。为了阻止这场疯狂的游戏，决定打破底线。
- **圆桌讨论**: 玩家分享线索，进行第一轮推理和指控。

#### **第三阶段：第一次投票与揭示 (15分钟)**
- **第一次投票**: 玩家投票指认在“当前现实”（实际上是虚拟世界）中杀害Damian的凶手。
- **世界观反转与画外音**: 
    - 就在玩家投票选出凶手后，**Atlas的画外音突然在所有人耳边响起**。
    - **Atlas**: “测试结束。感谢各位的参与。刚才的一切，都是为了选拔最适合领导Davion未来的继任者而进行的虚拟现实模拟。”
    - **真相大白**: 玩家此时才意识到刚才经历的“现实”和“谋杀”都是假的。Damian并没有死，这只是一场极度逼真的测试。
    - **指令**: Atlas指示所有玩家退出虚拟连接，回到现实世界的会议室，它将在那里宣布测试结果以及最终的继任者。

---

### 第二幕：现实世界的谋杀 (约90分钟)

#### **第四阶段：回归现实与真实死亡 (20分钟)**
- **退出系统**: 玩家“醒来”，摘下设备，回到现实中的会议室。气氛一度从紧张转为困惑或释然。
- **意外发生**: 
    - Atlas正准备根据刚才虚拟测试的表现宣布继任者。
    - **突发状况**: 在Atlas宣告结果之前，它突然检测到异常——**Damian Ionis在现实中已经死亡**。
    - **死亡时间**: 就在所有人沉浸在虚拟世界进行测试的这段时间里，Damian在现实中被杀害了。
    - **Atlas的决定**: 这是一个完全意料之外的错误（System Error）。因为这个突发的真实死亡，Atlas决定**暂时中止继任者的宣告**。
    - **新任务**: Atlas宣布，在宣告继任者之前，必须先带领大家找出**现实世界中杀害Damian的真凶**。
- **气氛营造**: 从“以为是真其实是假”的虚惊一场，瞬间转变为“以为是假结果真的死了”的恐怖现实。

#### **第五阶段：最终调查 (50分钟)**
- **第二轮搜证**: 搜证范围扩大到现实世界。线索变得更加“物理”。
    - **线索内容**:
        - Damian的癌症诊断报告。
        - Damian修改过的遗嘱，透露出对继承人的真实想法（倾向Simon）。
        - Zach与Damian关于AI伦理的激烈争吵邮件。
        - Damian办公室内一些与“GIFT Track”相关的实体文件。
        - Babara房间里藏有的关于Will死亡的调查资料。
        - 各角色在“退出虚拟世界”和“发现真实死亡”之间的不在场证明。
- **圆桌讨论**: 结合第一轮虚拟世界中被激发的“杀机”，以及第二轮的现实线索，进行最终的逻辑链梳理。凶手需要为第一轮的动机和第二轮的行动进行辩解。

#### **第六阶段：最终投票与结局 (20分钟)**
- **最终投票**: 玩家投票指认真正的凶手。
- **公布真相**: DM揭晓真正的凶手（Zach），并阐述其动机：阻止Damian继续放任Atlas进行疯狂的伦理实验，是为了“拨乱反正”。
- **结局演绎**:
    - **Atlas 的判决**: 继承人为 **Simon**。
    - **角色结局**:
        - **Damian**: 死前已选定Simon为唯一的“干净变量”。他试图以此否定Atlas的越界，但未及目睹结果。
        - **Alex**: 主动离开Davion，隐居避世。
        - **Adrian**: 试图做空公司被Simon检举，入狱。
        - **Joe**: 被Simon彻底清算，驱逐出集团。
        - **Nathan**: 退休，不再参与纷争。
        - **Zach**: 若被捕则入狱；若逃脱（未被检举）则辅佐Simon重构AI伦理。
        - **Babara**: 得知真相后释怀，退居幕后。
        - **Simon**: 继承公司。早年英明，晚年重蹈覆辙导致破产。
          *(特殊分支: 若Simon失去资格，Adrian在Alex退出后继承)*

---
### 第三幕：复盘
- DM带领玩家复盘整个故事，解答所有疑问，揭示所有未解的谜团和伏笔。
- 讨论游戏的核心主题，让玩家思考“程序正义”与“个体牺牲”的伦理困境。
