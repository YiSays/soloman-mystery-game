# 地图概念设计 (Map Concept Design)

本游戏包含两张地图。
一张是玩家在 **第一幕** 看到的 **虚拟 Davion 总部**。
另一张是玩家在 **第二幕** 醒来后所处的 **现实会议中心**。

---

## 地图 A：虚拟 Davion 总部 (The Virtual HQ)
**风格**：极简主义科幻风格，冷色调，充满未来感的玻璃和金属材质。所有物品看起来都过于完美，甚至有些失真（作为伏笔）。

### 布局结构
这是一个类似“回”字形的结构，中间是核心办公区。

1.  **CEO 办公室 (案发地)**
    *   **位置**：走廊的最深处，正对电梯厅。
    *   **描述**：巨大的落地窗（可以看到城市的数字霓虹夜景）。一张巨大的黑木办公桌。Damian的尸体就在这里。
    *   **关键点**：只有一扇门。隔音极好。

2.  **大会议室 (出生点)**
    *   **位置**：走廊的另一端。
    *   **描述**：玩家“进入”虚拟世界时，就是坐在这里的长桌旁。
    *   **连接**：通向主走廊。

3.  **茶水间 (The Pantry)**
    *   **位置**：位于CEO办公室和大会议室之间的走廊中段。
    *   **描述**：这里有咖啡机、冰箱。
    *   **关键点**：Babara 在这里下毒。这里也是 Joe 藏剪线钳（垃圾桶）的地方。

4.  **服务器机房 / 技术间**
    *   **位置**：角落。
    *   **描述**：闪烁着蓝光的服务器机柜。
    *   **关键点**：Nathan 可能会来这里查日志。

5.  **走廊 (The Corridor)**
    *   **描述**：连接所有房间。
    *   **关键点**：监控探头位于走廊天花板，被 Joe 破坏。

---

## 地图 B：现实会议中心 (The Real Conference Center)
**风格**：高端但写实的商务装修。温暖的灯光，但带有医疗设施的冰冷感。
**核心空间关系**：现实空间比虚拟空间小得多，因为虚拟空间是扩展的。

### 布局结构

1.  **中央会议室 (The Nexus)**
    *   **描述**：放着 8 张带有头部连接装置（Neural Link Chairs）的躺椅，呈环形摆放。
    *   **状态**：所有玩家醒来时都躺在这里。Zach 的位置离门口最近。

2.  **医疗休息室 (Medical Bay)**
    *   **位置**：紧邻中央会议室，只有一墙之隔。有一扇隔音门相连。
    *   **描述**：原本是为 Damian 准备的休息室，被改造成了临时重症监护室。
    *   **尸体**：Damian 在现实中死在这里的病床上。
    *   **关键点**：Zach 只需要走 5-10 步就能从会议室到达这里。

3.  **洗手间**
    *   **位置**：会议室外侧。
    *   **关键点**：Babara 可能会去这里冲掉毒药。

---

## 空间对应关系 (The Overlay)
*   **虚拟的大会议室** 对应 **现实的中央会议室**。
*   **虚拟的 CEO 办公室** 在空间上并不存在于现实会议室旁边（现实里那是墙），但在逻辑上，它对应的是现实中 Damian 所在的 **医疗休息室** 的位置。
*   **暗示**：当玩家在虚拟世界走向“CEO办公室”时，其实在潜意识里是走向“父亲所在的地方”。
