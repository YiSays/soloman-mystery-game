# 玩家规则手册 (Player Rulebook)

欢迎来到 **Davion 集团继承人选拔会议**。
本手册将指导您如何参与今天的会议与调查。

---

## 1. 游戏目标 (Objectives)
今天的核心任务是选出最有资格领导 Davion 的继承人。
然而，随着突发事件（Damian先生的死亡）的发生，您的首要目标变更为：
1.  **找出真凶**：通过搜集线索和逻辑推理，锁定杀害 Damian 先生的凶手。
2.  **隐藏秘密**：每个角色都有不为人知的过去或动机。请根据您的角色剧本，保护好自己的秘密。
3.  **达成个人目标**：完成您剧本中列出的特定任务（如争夺继承权、复仇、或保护某人）。

---

## 2. 游戏流程 (Game Flow)

### 阶段一：搜证 (Investigation)
在这一阶段，您可以在 Davion 总部内自由移动并寻找线索。
*   **行动点数 (AP)**：每位玩家拥有 **3点** AP。
*   **搜查**：
    *   搜查一个**公共区域**（如走廊、茶水间）消耗 **1 AP**。
    *   搜查一个**私人区域**（如某人的房间、搜身）消耗 **1 AP**。
    *   深入调查（需要特定技能或道具）可能会消耗额外 AP。
*   **线索卡**：当您消耗 AP 搜查某处时，主持人(DM)会发给您对应的线索卡。有些线索是公开的，有些是可以私藏的（具体询问DM）。

### 阶段二：圆桌讨论 (Discussion)
*   **公聊**：所有人在会议桌前进行公开辩论。分享您发现的线索，质问嫌疑人。
*   **私聊**：在特定环节，您可以邀请其他玩家进行一对一的私密谈话。这是结盟、交换秘密或进行交易的好机会。

### 阶段三：投票 (Voting)
*   游戏结束前，系统将启动投票程序。
*   您需要票选出您心目中的**真凶**。
*   票数最多者将被“检举”。若检举正确，平民阵营获胜；若检举错误，凶手逃脱。

---

## 3. 游戏礼仪与注意事项 (Etiquette)
1.  **角色扮演**：请全程保持在角色状态中（In-Character）。用角色的语气说话，思考角色的动机。
2.  **绝对保密**：严禁向其他玩家直接展示您的剧本。您可以用自己的话复述剧本内容，但不能直接给别人看。
3.  **诚实原则**：对于您获得的**线索卡**内容，您可以选择隐瞒或撒谎，但不能伪造线索卡本身。
4.  **听从系统指令**：Atlas 系统（DM）拥有最高解释权。
