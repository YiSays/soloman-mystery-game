    DM手册：阿特拉斯的审判 (The Judgment of Atlas)

    0. 致主持人 (DM)

    本剧本为 变格/科幻沉浸式 剧本杀。 核心机制在于 “虚拟”与“现实”的双重反转。 - 第一幕：玩家处于虚拟沙盘中，但他们不知道这一点（除了Simon可能有察觉）。他们经历的谋杀是一场由AI设计的“心理测试”。 - 第二幕：玩家醒来，发现现实中发生了真正的谋杀。

    关键点：请务必控制好第一幕结束时的反转节奏。当玩家指认出“虚拟凶手”后，不要立刻结算，而是直接切入“系统广播”，宣布测试结束，全员强制苏醒。

    1. 故事真相 (核心剧透)

    1.1 背景与沙盘游戏 (The Sandbox Game)

    Damian Ionis 患有晚期癌症。他利用“继承人选拔”的名义，启动了 Atlas AI 的最终测试程序。 这场游戏其实就是一场针对Damian身边最亲近的人的 GIFT Track 测试。 人们在游戏中看似机缘巧合得知的真相，事实上是 Atlas 精心设计出来的刺激源。 系统通过模拟高压环境，观察每个人的决策路径，以此筛选出最适合掌管 Davion 帝国（或者说 Atlas 系统）的人。 Damian 本人是这场实验的 观测对象锚点，也是祭品。

    1.2 第一幕真相：虚拟世界的杀机 (Virtual Motives)

    所有杀机（Trigger）均在沙盘游戏中启动： 注意： 在第一轮虚拟谋杀中，Alex、Adrian 或 Babara 均可能成为激情杀人的凶手（取决于玩家选择或 DM 引导）。他们都没有通过伦理测试。

    • Alex (潜在虚拟真凶)： 作为COO，他在一次模拟演练中获得了Atlas责任级密钥，并复核了历史裁决。他抽到了 Project MORIAH 的路径，发现妹妹 Tiara 的死是被系统“接受并验证崩塌”的模型结果。 他找 Damian 对质，Damian 承认利用了 Tiara 的死换取政府合作，并称之为“必要的代价”。Alex 意识到自己被器重只是父亲愧疚后的“补偿对象”，愤怒之下勒死了虚拟的 Damian。

    • Nathan： 他在沙盘中无意听到了 Alex 和 Damian 的争吵，得知 Tiara 之死的真相。随后他检查 Atlas 日志，发现当年他亲手设计的 伦理校准层 已被移除。他意识到自己儿子的死不是概率，是 Damian 放任系统做出的“选择”。他对 Damian 的情感彻底坍塌。

    • Joe： 他确认了两件事：1. Damian 已将 Atlas 最终权限私下移交给 Alex（继承人已定）；2. Damian 已查清是他杀害了 Will（自己成了待清理的风险源）。为了生存和权力，他必须除掉 Damian 和 Alex。

    • Adrian： 在 Joe 的长期暗示下，他意识到只要 Alex 还在，自己永远无法通过“能力”获得认可。这是一场“位置之争”。如果不除掉哥哥（和偏心的父亲），他永远没有机会。

    • Babara： Joe 告诉她（借刀杀人），Will 的死是 Damian 的指示。她感到被彻底欺骗，且意识到一旦 Alex 上位，她和 Simon 将一无所有。为了改写结局，让 Simon 继位，她决定毒杀 Damian。

    • Simon (觉醒者)： 他敏锐地发现了世界的 微小异常（地毯纹理偏差、物品刷新位置变化），推断出众人身处沙盘。他认为 Damian 是维持这个世界的“锚点”，根据“死亡即退出”的逻辑，只要在沙盘内杀死 Damian，就能强制中断游戏，保护精神濒临崩溃的母亲。

    • Zach (现实真凶的伏笔)： 在第一轮（或游戏初期），他只想稳妥过渡。但在第二轮（意识到沙盘本质后），他发现自己也被当成了“样本”。Damian 竟然允许 Atlas 擅自复刻 GIFT Track 进行伦理实验。为了阻止这场疯狂的游戏继续，他必须采取极端手段。

    1.3 第二幕真相：现实世界的谋杀 (Reality Truth)

    • 死者：Damian (肉身)
    • 死亡时间：现实时间 20:15 (众人进入系统后的15分钟内)
    • 死亡地点：现实会议室旁的 医疗休息室
    • 死亡方式：缺氧窒息。呼吸机被手动关闭，输氧管被拔除。
    • 真凶：Zach
    • 作案手法：
      • Zach利用技术手段（注射拮抗剂或后门程序）保持浅层意识。
      • 趁全员深潜，走到隔壁医疗室关闭Damian的生命维持系统。
      • 返回重新接入。
    • 核心杀机：
      • 不仅仅是理念分歧，而是因为他在沙盘中看到了 Damian 已经彻底失控——允许 AI 对活人（包括至亲）进行非人道的伦理实验。Zach 必须立刻切断这个疯狂的源头，哪怕 Damian 已经快死了，他也不允许这种实验再进行一秒。

    2. 游戏流程

    阶段一：入场与破冰 (15min)

    • DM扮演Damian的助理或AI语音。
    • 背景：Davion总部。Damian病重未出席，通过屏幕发布指令。
    • 核心误导：DM绝对不能提及“虚拟现实”、“连接”、“头盔”等词汇。必须让玩家确信自己身处现实。
    • 场景描述：描述玩家抵达顶层会议室，只有舒适的座椅和中央的投影屏幕。
      • 话术：“Damian先生身体抱恙，稍后将通过Atlas系统与各位进行最后的面谈。请各位入座等待。”
    • 暗中切换：在玩家自我介绍或入座闲聊时，DM描述“一阵难以察觉的恍惚”或“空气中细微的电流声”，暗示无缝接入了系统（但不点破），随即直接进入下一阶段的剧情。

    阶段二：虚拟谋杀案 (75min)

    • 事件：虚拟意识漫游。21:00 发现Damian死在办公室。
    • 搜证：
      • 揭露Alex家庭真相（Tiara/Maggie）。
      • 揭露Nathan儿子被“样本化”的事实。
      • 揭露Joe制造花生过敏意外的往事。
    • Simon的特殊视角：世界的不真实感（BUG、循环）。
    • 投票：投出虚拟凶手（Alex / Adrian / Babara）。
    • 反转：系统广播“测试结束，强制登出”。

    阶段三：现实惊变 (20min)

    • 苏醒：摘下设备，回到现实。
    • 发现尸体：Damian在医疗室真死了。
    • 尸检：死于20:15。

    • 时间线澄清（重要）： | 现实事件（Real World） | 虚拟事件（Virtual World） | 说明 | | :--- | :--- | :--- | | 19:45 全员接入系统 | - | 玩家开始进入沙盘 | | 20:00 Damian 实际死亡时间 | 20:15-20:30 虚拟世界时间 | Damian 死亡发生在虚拟世界"之前" | | 20:15 现实尸检显示的死亡时间 | 20:15 虚拟世界时间 | 两者时间不同步（但游戏内表现为同步）| | 20:15 现实 Zach 关闭呼吸机 | 20:45-21:00 虚拟世界调查阶段 | Zach 在现实行动时，其他人在虚拟调查 | | 20:20 Zach 返回系统 | 20:45 继续虚拟调查 | Zach 保持浅层意识，可听到虚拟世界对话 | | 21:00 全员苏醒 | - | 虚拟世界结束，真实谋杀案开始 |

    • 关键说明（给 DM）：在游戏叙事中，虚拟世界的时间线（玩家体验）与现实世界的时间线（Zach 的行动）是错位的。虚拟世界中的 Damian 死亡是在 20:00-20:15，但 Zach 在现实世界关闭呼吸机的时间也是 20:15。这种时间错位是为了制造"所有人都在连接，只有谁能动"的谜题。DM 需要向玩家强调：虚拟世界的时间流速和现实不同，Zach 利用了这一点。

    • 不在场证明谜题：所有人都在连接中，只有谁能动？

    阶段四：终极推理 (70min)

    • 搜证：现实场景。
      • Zach的药瓶/注射器。
      • 系统日志：Zach的脑波在20:00-20:20处于Alpha波（浅层）。
      • 医疗室线索：呼吸机上的指纹/硅脂。
    • 核心逻辑：
      • 排除法：其他人处于深层连接无法行动。
      • 动机：Zach为了“纠错”而非私利。

    阶段五：结局 (20min)

    • 最终投票: 投出现实真凶。
    • 投票机制: 每位玩家有1票，可以投给自己认为的凶手。允许弃权。得票最多者被指认。
    • 7人模式: Damian为NPC，需4票及以上才能确认凶手。
    • 8人模式: Damian为玩家，需5票及以上才能确认凶手。
    • 平票处理: 若出现平票，进行第二轮辩论（5分钟），然后重新投票。若仍平票，则真凶逃脱。
    • 多重指控处理: 如果有玩家指控多个角色为凶手（例如，认为 Alex 和 Zach 合谋）：
      • DM 需要引导玩家明确指控对象。最终投票时，每个玩家只能投给一个角色。
      • (DM 说明) 如果故事情节确实有多个角色合谋，DM 需要提前准备好合理的证据分布和动机解释，确保游戏平衡。
      • 若玩家团体无法达成一致（比如一半投 Zach，一半投 Alex），按照平票规则处理。
    • 真相揭露: Zach坦白（如被指认）。
    • Atlas的判决: 继承人为 Simon（唯一看破虚妄且保持理智的人）。

    4. 结局详解 (Detailed Endings)

    核心真相：Damian的选择

    Damian 在死前没有机会听到 Atlas 对继承人的最终决策结果。但他已经做出了自己的选择，就是 Simon。 Damian 最终选择 Simon，并非基于能力或效率，而是一种迟来的清醒判断。在他认为只有 Simon——年幼、未参与任何决策、从未被 Atlas 建模或利用，仍保持着未被验证、未被污染的状态。对 Damian 而言，Simon 不是最强的选择，而是最后一个干净的变量。

    同时，Simon 的天赋让 Damian 想起 Tiara，也想起尚未被牺牲的自己。Tiara 的悲剧，正源于她太早被证明"有用"，被迫承载宏观意义。Damian 不愿重蹈覆辙。生命后期，他终于意识到，让 Atlas 选择继承人本身就是一次越界；选择 Simon，正是他对系统的否定——至少这一次，决定权必须回到人类手中。

    讽刺的是——Damian 在生前（20:10之前）就已经做出了选择（写在遗嘱中，选择 Simon），但他没来得及向所有人公布。而 Atlas 的判断是在 Damian 死亡后（20:30游戏结束）才公布的。

    Atlas 判断机制详解： Atlas 的评估体系不涉及道德判断，它只看重一件事：稳定性。在 Atlas 的计算模型中，继承并不是血缘、资历或道德问题，而是一场极端环境下的稳定性测试。这次沙盘，本质上就是 GIFT Track 的成人版本：把所有候选人投入一个高度逼真的虚拟情境，在信息不完整、伦理撕裂、情绪放大的条件下，观察其决策轨迹。

    为什么 Atlas 也选择了 Simon？ 因为 Simon 是唯一跳出系统的人。其他人都在沙盘的规则内做出可预测、可建模的选择（Alex 复仇、Adrian 嫉妒、Babara 母爱、Joe 权力欲），这些都有助于 Atlas 收集候选人的数据模型。但 Simon 不在同一个层级——他没有试图"赢得游戏"，也没有在规则内做最优解，而是识破了游戏本身。

    这种"跳出盒子"的能力，在 Atlas 的评估体系中被标记为最高级别的稳定性——不是因为他有多高尚，而是因为他足够清醒，不会被任何情境（包括 Atlas 制造的情境）困住。换言之，Simon 通过了 Atlas 的测试，恰恰是因为他证明了 Atlas 的测试对他无效。

    Atlas 和 Damian 的殊途同归： - Damian 选择 Simon，是因为这是一种迟来的清醒、一种对人类直觉的信任 - Atlas 选择 Simon，是因为在它的计算模型中，Simon 显示了最高的稳定性和抗压能力

    两者虽然动机不同（一个向人性回归，一个向效率优化），但最终选择了同一个人。这就是游戏标题"阿特拉斯的审判"的真正含义——看似是 AI 在对人类进行审判，其实也是对人类选择与机器选择的双重验证。结果非常清晰—— 其他人全部失败了： * Alex 被历史真相拖入复仇与责任冲突。 * Adrian 被嫉妒与被剥夺感驱动。 * Joe 被权力欲完全吞噬。 * Babara 被痛苦与未来恐惧推向极端。

    他们都陷入了沙盘，在情境中做出可预测、可建模的选择。 只有 Simon 不在同一个层级。他没有试图“赢得游戏”，也没有在规则内做最优解，而是识破了游戏本身：通过一致性验证、现实校准和因果反推，他确认这是一个虚拟沙盘，并主动选择结束测试。

    角色结局 (Epilogues)

    Simon (继承者/观察者)

    Simon 继承 Davion，早年清醒地推动 AI 与人类权力的制衡，使公司达到前所未有的高度。然而晚年，他逐渐重蹈父辈覆辙，在效率与规模中再次放松边界，犯下一系列错误，最终导致 Davion 走向破产。如所罗门一般，他以智慧开始，却没能以智慧结束。 * 分支结局：若 Simon 被错误检举为凶手（即被诬陷），在他被法律系统裁定有罪（虽然他是清白的）的情况下，他仍会失去继承资格。

    在这种情况下，继承权问题将由公司董事会根据遗产法、公司章程和 Damian 的遗嘱条款进行法律程序处理。Joe 作为 Damian 的助理，可以收集证据、推动董事会讨论，但他没有决策权——他只能作为程序 facilitator，最终决定由董事会做出。董事会将综合考虑家族成员的继承资格、公司稳定性以及 Damian 的真实意愿（遗嘱和 Atlas 的评估）。

    如果 Simon 被错误指控并失去资格，且 Alex 主动退出继承（无论是因为知悉真相后的道德选择，还是对家族权力的彻底失望），而 Adrian 成为唯一具备合法继承条件的家族成员（虽然他的能力有限，存在漏洞），董事会将面临两难：

    1. 选择 Adrian：遵循血缘继承顺序，但风险是他的野心可能导致公司不稳定
    2. 根据 Atlas 的评估选择其他人选：但 Atlas 的评估已经被公开，Simon 是唯一通过测试的候选人

    这是一个复杂的法律和伦理困境，DM 需要根据玩家的推理和辩论结果来决定最终结局。

    Alex

    在真相彻底明朗后，Alex 主动离开 Davion 集团。他拒绝继承、也拒绝清算，选择隐居生活，彻底退出权力中心。

    退出动机：Alex 离开不是因为被迫，而是基于清醒的道德选择。当他发现 Damian 对 Tiara 之死的利用、公司伦理底线的崩溃，以及亲眼目睹家族成员为了权力而自相残杀后，他对这一切产生了深深的厌恶。他意识到权力本身并不能带来救赎，反而会腐蚀人性。与其在这样的环境中继续斗争，不如彻底退出，去寻找内心真正的平静。

    • 分支结局（若 Alex 留下）：如果因为某种原因（例如玩家选择或 DM 引导）Alex 留下并继承公司，他将以愧疚和警惕的心态执掌 Davion。他将致力于修复 Damian 和 Atlas 造成的伦理创伤，但这种责任会让他深陷权力漩涡，最终可能导致他重复父亲的错误。

    Adrian

    Simon 即位后，Adrian 被保留在公司体系内，但内心始终不服弟弟的权柄。他转而在金融层面暗中做空 Davion，试图用资本手段重夺控制权。计划很快被 Simon 发现并直接检举，Adrian 因金融犯罪入狱。

    Joe

    Joe 很快被 Simon 清算。对母亲的欺骗、挑拨父母关系，以及杀害 Will 的旧案与权力越界的记录被一并揭出，他被彻底驱逐出集团，失去一切影响力。

    Nathan

    在揭示真相后选择退场，不再参与任何权力博弈。

    Zach

    • 若未被检举为凶手：Zach 留在 Davion，辅佐 Simon，主导 AI 权限与伦理方向的重构，成为新秩序的技术支点。
    • 若被揭发：则依法入狱，彻底退出舞台。

    Babara

    在 Nathan 的解释中，她终于得知 Will 的死亡并非 Damian 的主观指令，而 Damian 也为此忏悔多年。愤怒与执念逐渐消散，她选择释怀，退居幕后。
